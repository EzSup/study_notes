# Virtual dom
Virtual dom - копія реального dom-дерева, яка існує в пам'яті. Потрібен для відстеження змін. І ререндера тільки тієї частини реального DOM-дерева, яка реально змінилась.
Насправді React містить 2 копії віртуал дома, до і після змін, і їх порівнює. А не реальний дом.
# Hooks
Хуки - це спеціальні функції в React, які довзоляють використовувати стан і життєвий цикл компонентів без класів.
